# Agents Configuration

이 파일은 workfolio 프로젝트들에서 공유하는 에이전트 설정을 포함합니다.

## 사용 가능한 에이전트

### Planner

복잡한 기능 구현 및 리팩토링을 위한 기획 전문 에이전트입니다.

| 항목 | 내용 |
|------|------|
| **파일** | [planner.md](agents/planner.md) |
| **도구** | Read, Grep, Glob |
| **모델** | opus |
| **활성화** | 기능 구현, 아키텍처 변경, 복잡한 리팩토링 요청 시 |

**주요 기능:**
- 요구사항 분석 및 상세 구현 계획 수립
- 복잡한 기능을 관리 가능한 단계로 분해
- 의존성 및 잠재적 위험 식별
- 테스트 전략 및 롤백 계획 포함
